# Scale Isomorphism

This note explains why the reprimand feedback loop dataset exists as a micro-scale counterpart to the sanctions feedback loop dataset.

## Macro Case: Sanctions

At the macro scale, sanctions impose pressure on states, regimes, logistics networks, financial systems, and local economies.

Expected linear model:

```text
pressure increases
→ cost increases
→ compliance increases
```

Observed nonlinear possibility:

```text
pressure increases
→ local adaptation
→ shadow networks
→ resilience narratives
→ regime hardening or delayed reversal
```

## Micro Case: Reprimand

At the micro scale, reprimand imposes pressure on a child, student, worker, or small group.

Expected linear model:

```text
reprimand increases
→ discomfort increases
→ correction increases
```

Observed nonlinear possibility:

```text
reprimand increases
→ threat narrative
→ masking or silence
→ defiance consolidation / self-efficacy erosion
→ behavioral persistence or delayed fracture
```

## Shared Structure

Both cases can be abstracted as:

```text
external pressure
→ adaptive system
→ hidden internal state change
→ delayed nonlinear response
```

The purpose of the comparison is not moral equivalence.

The purpose is structural observation.

The micro case makes the macro case body-grounded.
The macro case makes the micro case structurally visible.
