# Observation Model

This repository observes reprimand as pressure entering an adaptive system.

## Core Variables

- external pressure
- public exposure
- perceived threat
- self-efficacy erosion
- defiance consolidation
- silence / masking
- behavioral persistence

## Observation Target

The target is not behavior alone.

The target is the gap between visible behavior and hidden adaptation.

A visible correction may coexist with internal fracture.
A visible silence may conceal resistance.
A visible defiance may indicate loss of recoverable trust.

## Key Structural Question

```text
When does pressure stop correcting behavior
and begin changing the internal state of the system?
```

## Nonlinear Risk

Pressure may produce short-term compliance while increasing long-term instability.

This dataset attempts to make that transition observable.
